2. 由于QHPW是六代es，核显为HD530，所以仿冒为了HD630，如果你用了七代U并且不会修改，请发Issues问我
3. 目前除了睡眠全部完美，网卡使用Intel AX200
4. OpenCore版本应该是0.8.9开发版，支持最新macOS 13.1 Ventura。
